# ByeFinder
Close finder on macos with the ByeFinder app.

## Download
```
curl -L https://raw.githubusercontent.com/vutondesign/ByeFinder/master/ByeFinder.zip
```

## Git Clone
```
git clone https://github.com/ktrvs/ByeFinder
npm install
```
